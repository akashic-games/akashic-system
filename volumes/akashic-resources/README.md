# akashic-resources

リソース置き場です。Akashic コンテンツの起動に必要なものを配置します。

## contents

content.json を配置します。

### 配置

```
contents/<gameCode>/<revision>/
```

## engines

コンテンツの実行に必要なエンジンを配置します。

### 配置

```
engines/<engineCode>/<revision>/
```

## games

コンテンツそのものを配置します。

### 配置

```
games/<gameCode>/<revision>/
```
