# system-control-api-client

SystemAPI 関連の各内部 API を利用するためのクライアント。

## ログ出力方針

秘匿情報が関係するため、下記の方針を適用しています。

- 全体的に秘匿情報が関係する Client には logger を渡せないようにする
- メソッドごとに出力を切り替えることは考えていない
  - 2015/11/19 時点
  - 考えたほうがよいと判断したときに実装方針を考える
