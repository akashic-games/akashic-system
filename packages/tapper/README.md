# tapper

dapper like typescript ORM

### コンパイルオプションについて

TypeScript の [decorator](http://www.typescriptlang.org/docs/handbook/decorators.html) と [reflect-metadata](https://github.com/rbuckton/reflect-metadata) を利用しているため、 `experimentalDecorators` と `emitDecoratorMetadata` を有効にする必要があります。
