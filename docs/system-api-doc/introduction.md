# Akashic System API

## 概要

Akashic システムを利用するための API です。Akashic Engine 製のコンテンツをサーバで実行し、結果を得ることができます。

代表的な利用方法に「[ニコ生ゲーム](https://akashic-games.github.io/shin-ichiba/)」があります。また、生放送のギフト機能やクルーズ・ニュース番組などは「[共体験フレームワーク](https://github.com/akashic-games/coe/blob/master/getstarted.md)」を使ったコンテンツで実現しています。
