# プレーログ API

## GET /plays/:id/playlog

### 概要

プレーログの取得を行います。

**この API はシステム負荷の高い操作になります。** 対象プレーログの規模に比例してレスポンスタイム・サイズも増大するため、利用に際してはご注意ください。

### URIパラメータ

| キー | 型     | 必須 | デフォルト | サンプル | 備考                               |
| ---- | ------ | ---- | ---------- | -------- | ---------------------------------- |
| id   | BigInt | ◯    |            | 1        | プレーログを取得する対象のプレーID |

#### リクエスト

```sh
curl ${ENDPOINT}/v1.0/plays/1/playlog
```

### クエリパラメータ

| キー     | 型     | 必須 | デフォルト | サンプル  | 備考                                   |
| -------- | ------ | ---- | ---------- | --------- | -------------------------------------- |
| excluded | string |      |            | ignorable | 取得するplaylogから除外するeventを指定 |

#### リクエスト

```sh
curl ${ENDPOINT}/v1.0/plays/1/playlog?excluded=ignorable
```

### レスポンス

| キー | 型     | 概要                                                           | 備考                                                                                                                                                  |
| ---- | ------ | -------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| data | String | プレーログのデータを MessagePack + Base64 エンコードした文字列 | [staticPlaylogWorker](pecification_parameters.md#code-staticplaylogworker) の playData または、 [POST plays](plays.md) のデータとしても利用できます。 |

 

#### 成功

```json
{ "meta": { "status": 200 }, "data": "gqh0aWNrTGlzdJIAzI2rc3RhcnRQb2ludHORgqVmcmFtZQCkZGF0YYGkc2VlZMtCdaGcr+3gAA==" }
```

#### 失敗

| ステータスコード | エラーコード          | 備考                       |
| ---------------- | --------------------- | -------------------------- |
| 400              | INVALID_PARAMETER     | リクエストパラメータが不正 |
| 404              | NOT_FOUND             | プレーが存在しない         |
| 500              | INTERNAL_SERVER_ERROR | システムエラー             |
