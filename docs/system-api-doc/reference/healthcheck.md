# ヘルスチェック API

## GET /healthcheck

### 概要

Akashic システムが稼働中であるかのヘルスチェックを行います。
レスポンスが成功以外を示す場合、Akashic システムの他の API も正常に動作しない状態にあります。

成功以外を返す一例:

- **システムが稼働していない**
  - 多くの場合は `HTTP status code: 404` を返します。
  - メンテナンス時間外にこの状態になるとき、Akashic システムそのものがダウンしている可能性があります。
- **一時的にレスポンスを返せない**
  - 多くの場合は `HTTP status code: 504` を返します。
  - API, DNS, LB, Proxy などの一時的な負荷によってこの状態になることがあります。
  - リトライによって成功することがあります。リトライ回数についてはそれ自体が負荷にならないようご注意ください。

### リクエスト

この API のエンドポイントはバージョン文字列 (`/v1.0/` など）を含めません。

```sh
curl ${ENDPOINT}/healthcheck
```

### レスポンス

この API はレスポンス Body に有意な値を返しません。 HTTP status code のみを取得してください。

```sh
curl ${ENDPOINT}/healthcheck -o /dev/null -w '%{http_code}\n' -s
200
```

| ステータスコード | エラーコード | 備考          |
| ---------------- | ------------ | ------------- |
| 200              | 成功         |               |
| 200 以外         | 失敗         | 404, 504 など |
